deploy

  • Save changes to localauthor.org file (or any config files in /localauthor); on save, ox-hugo should automatically export changes in localauthor.org to md files in directory /content.
  • Execute ./deploy.sh (in /localauthor); it should run ‘hugo -D’ to build site in /public, then it should automatically commit/push changes (to public file) to localauthor.github.io (your public git repo and public site)
  • You can then commit and push changes (to localauthor.org itself or files in /localauthor) to localauthor.git (your private repo); you don’t have to commit/push here before you deploy
