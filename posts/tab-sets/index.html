Taming Emacs Tab Clutter with `tab-sets.el`

——————————————————————————————

Emacs offers a wealth of options for managing your workspace: frames, windows, tabs, workspaces, perspectives, projects… the list goes on. Here’s another tool for the pile: tab-sets.

As is common, this package emerged to address a specific need. I rely heavily on `tab-bar-mode`, using tabs in Emacs frames much like browser tabs. Each tab in a frame typically holds a file essential to my current project. The trouble was, I couldn’t find a reliable way to save and restore tabs. And if one exists that I didn’t find, I don’t want to know about it now… :)

My primary use-case for `tab-sets` is for teaching and writing.

I often teach the same class several times a week. Before each session, I found myself repeating the process of finding and re-opening the files required for that class: lecture notes, slides (exported from Org-reveal), and related materials.

For writing projects, the situation was similar. I would keep a dedicated frame with tabs for drafts and notes for quick access. Setting this up each time felt inefficient.

`tab-sets.el` addresses this by allowing you to save and restore the tab configuration of your `tab-bar-mode` frame. Saved tab-sets persist across Emacs sessions.

If you use `tab-bar-mode` to organize project files within frames, `tab-sets.el` might be useful. You can find it on

Usage

  1. Saving a tab-set: With the desired frame selected, `M-x tab-sets-save`. You’ll be prompted for a name.
  2. Opening a tab-set: `M-x tab-sets-open`. Select the tab-set to open. By default, it opens in a new frame. Use a prefix argument to open in the current frame.
  3. Deleting a tab-set: `M-x tab-sets-delete`.
  4. Renaming a tab-set: `M-x tab-sets-rename`.

Tab-sets are also integrated with bookmarks, allowing you to use `bookmark-jump` to open them. Run `M-x tab-sets-reconcile-bookmarks` to ensure bookmarks are created for all saved tab-sets.

For Embark users, `(tab-sets-setup-embark)` adds Embark actions for tab-sets. Marginalia users can use `(tab-sets-setup-marginalia)` for annotations in completion UIs.

Customization

  • `tab-sets-data-file`: File to store tab-set data. Set this before saving.
  • `tab-sets-same-frame`: Open tab-sets in the current frame by default (toggles with prefix argument in `tab-sets-open`).
  • `tab-sets-confirm-delete`: Confirm before deleting tab-sets.
  • `tab-sets-bookmark-store`: Enable/disable bookmark integration.
  • `tab-sets-bookmark-prefix`: Prefix for tab-set bookmark names.
